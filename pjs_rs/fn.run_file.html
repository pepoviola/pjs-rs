Run a js/ts code from file in an isolated runtime with polkadotjs bundles embedded

The code runs without any wrapping clousure, and the arguments are binded to the pjs Object, available as pjs.arguments. If you want to to use top level await you will need to wrap your code like we do in run_file_with_wrap.
